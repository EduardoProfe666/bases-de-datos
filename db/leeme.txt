Para poder instalar la base de datos requerida para ejecutar la aplicación necesita Postgres 9.5.
Puede proceder de 2 formas para instalar la base de datos requerida:
1-. Restaurar la base de datos con el backup proporcionado, el cual contiene datos para probar la aplicación. El backup no contiene el script de creacion del DataBase, por lo que primero debe crear un database y restaurar el backup en el mismo.
2-. Crear un database y ejecutar las sentencias sql que se proporcionan. La primera permite inicializar la base de datos con todas la tablas necesarias. La segunda dota a la base de datos de todas las funcionalidades requeridas. La tercera dota a la base de datos con los triggers necesarios. La cuarta, la cual es opcional, dota a la base de datos con datos de prueba. Tenga en cuenta que si no se ejecuta la cuarta, debe crear manualmente en la base de datos al menos un usuario administrador, el cual pueda crear un secretario docente, el cual a su vez pueda crear estudiantes.
